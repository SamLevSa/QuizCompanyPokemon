# WebQuiz The Company Pokemon

## Sobre o Quiz
Esse Website é um Quiz Interativo sobre a The Company Pokemon, desenvolvido como atividade acadêmica.
Foi feito utilizando HTML5, CSS3 e Javascript puro, sem frameworks.

## Instruções de Execução

### Pré requisitos
Navegador Moderno (Chrome, Firefox, Edge etc).
Nenhum programa ou recurso externo é necessário.

---

### Etapas
1. Baixe o repositório (Via Git ou Zip).  
##### 1.1 - Baixando pelo zip:
No repositório clique em:  
```
<>Code 
```

Depois clique em:  
```
Download Zip
```


Agora é só extrair a pasta.

---
##### 1.2 - Baixando pelo terminal do Git:  
(No terminal do git) clone o repositório:  
```
git clone https://github.com/SamLevSa/QuizCompanyPokemon.git
```

Depois entre na pasta clonada:  
```
cd QuizCompanyPokemon
```

E então execute:  
```
explorer .
```

---
2. abra o arquivo ```index.html``` .

(Se perguntar qual aplicativo usar pra abrir, abra com o seu navegador preferível.)

## Instruções de uso

Após abrir o arquivo, você entrará no site do quiz interativo. 
   
Configure seu temporizador clicando nos botões de mais(+) ou menos(-).  
Após definir o tempo desejado, clique em ``` começar quiz ``` .  

Leia a pergunta apresentada e escolha uma das 5 alternativas *(A, B, C, D e E)* disponíveis.  
Depois clique em ``` Próxima pergunta ``` para prosseguir.  

Após terminar todas as perguntas, clique em ``` Finalizar Questionário ```.

Ao final o sistema exibirá sua pontuação total e um gabarito para comparação das suas repostas com as corretas.
